# Changelog

All notable changes to Alfaifi Model Advisor are documented in this file.

The project follows [Semantic Versioning](https://semver.org/).

## [Unreleased]

## [0.2.0] - 2026-07-22

### Added

- Post-interview discovery across the complete official Ollama library.
- Goal-aware shortlisting and live verification of relevant family tags.
- Conservative profiles for newly discovered official families.
- Animated model-discovery waiting screen with visible progress messages.

### Changed

- Redesigned terminal header, numbered workflow sections, hardware card, result
  hierarchy, and project footer.
- Project links now display their complete copyable URLs in every terminal.
- Unknown licenses are labeled explicitly and excluded by the permissive-only
  filter.

## [0.1.0] - 2026-07-22

### Added

- Private Windows hardware inspection for CPU, RAM, GPU, VRAM, storage, and
  Ollama.
- Interactive model recommendation wizard with task, language, quality,
  context, privacy, vision, tool-calling, and license preferences.
- Explainable scoring for trusted Qwen, Gemma, and Kimi model entries.
- Official-source refresh, 24-hour cache, and offline fallback catalog.
- Approval-gated Ollama installation with strict model identifier validation.
- Human-readable and JSON command output.
- Graphical per-user Windows installer and portable release package.
- Apache-2.0 source license and independent trademark policy.
